# 📊 UK Crime Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on large-scale UK police crime data to uncover trends, patterns, and insights across time, location, and police forces.

# 🚀 Project Overview

Dataset contains 14+ million crime records (2023–2025)

Focus on temporal, spatial, and institutional analysis

Identifies crime trends, seasonal patterns, and hotspots

# 🔍 Key Analysis Performed
## 📅 Time Trends

Yearly and monthly crime patterns

Seasonal and quarterly analysis

## 🗺️ Geospatial Analysis
Crime density heatmaps

Urban vs rural clustering

## 🏢 Police Force Analysis
Crime distribution by force

Outcome efficiency comparison

## 📈 Outcome Analysis

Case resolution distribution

Investigation success rates

## 🧹 Data Cleaning
Removed missing values and duplicates

Validated geographic coordinates

Converted date features (Year, Month)

Ensured data consistency for analysis

## 📊 Key Insights
Crime shows seasonal peaks in summer (June–July)

Majority of cases result in:

No suspect identified

Unable to prosecute

High crime concentration in urban areas (e.g., London)

Overall slight decline in crime from 2023 to 2025
